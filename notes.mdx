| command                                | desc                                          |
| -------------------------------------- | --------------------------------------------- |
| `./reach init`                         | _*scaffold project*_                          |
| `./reach config`                       | _*configure connector (ETH or Algo)*_         |
| `./reach compile`                      | _*compile project and build contract*_        |
| `./reach down`                         | _*solve container conflict*_                  |
| `./reach run`                          | _*run backend*_                               |
| `REACH_CONNECTOR_MODE=foo ./reach run` | _*run with ETH, ETH-live or ALGO, ALGO-live*_ |
